Тонкий клиент запущен на на хосте с ubuntu/windows, на вм стоит ubuntu
1) В гостевой вм установить spice-webdavd service командой sudo apt install spice-webdavd
Запустить сервис командой sudo spice-webdavd -p 8000
2) Подключить тонкий клиент к вм. В меню тонкого клиента "Настройки" тыкнуть "Общие папки". В открывшемся окне выбрать папку на хосте, которую хотим расшарить. Поставить галочку "Общая папка". Закрыть окно.
3) В сетевых папках вм появится пункт Spice client folder. Тыкнуть на него, чтобы открыть расшаренную папку.

Тонкий клиент запущен на на хосте с ubuntu, на вм стоит windows
1) В гостевой вм установить spice-webdavd-x64-latest (https://www.spice-space.org/download/windows/spice-webdavd/)
2)Открыть командную строку, перейти в папку "C:\Program Files\SPICE webdavd" и выполнить скрипт map-drive.bat
3) Подключить тонкий клиент к вм. В меню тонкого клиента "Настройки" тыкнуть "Общие папки". В открывшемся окне выбрать папку на хосте, которую хотим расшарить. Поставить галочку "Общая папка". Закрыть окно.
4)Появится новый сетевой диск Spice client, на котом и будет содержимое расшаренной папки
